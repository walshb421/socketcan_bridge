# Automotive Simulation Harness (ASH)

This tool is an **automotive network emulation and simulation bridge** — a scriptable, client-configured framework for interacting with, simulating, and testing automotive networks.

Can also be referred to as autosim harness, as it consists of two layers/planes, the harness for configuring the simulation, interfaces and mapping, and the autosim, the interaction layer that application signals are simulated/emulated.
